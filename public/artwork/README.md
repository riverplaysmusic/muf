# Artwork

This folder contains visual artwork for the label and releases.

## How to Add Artwork

1. **Drop your image here** - Just save it in this folder
2. **Name it clearly** - Examples:
   - `hero-artwork.jpg` (main homepage artwork)
   - `album-neon-dystopia.jpg` (album cover)
   - `book-fear-loathing.jpg` (book cover)
3. **Reference it in code** - Use `/artwork/filename.jpg`

## Recommended Specs

- **Format:** JPG or PNG
- **Size:** 1200x1200px or larger (square works best)
- **File size:** Keep under 2MB for fast loading
- **Quality:** High quality, this represents your label

## That's It

No database. No CMS. Just files. Braindead simple.

When you want to change the artwork, just replace the file and redeploy.
